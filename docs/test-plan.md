# Deskflow-Plus 测试计划

## 1. 测试目标

本测试计划覆盖：

- MVP 功能完成度测试
- Post-MVP 扩展能力测试
- 性能测试
- 安全测试
- 打包验证

所有任务在标记为完成前，都必须执行对应的功能测试。涉及实时输入、网络会话或平台行为的任务，还必须执行性能测试或记录“不适用”。

## 2. 测试矩阵

MVP 最低验证矩阵：

- Windows 主控 -> Windows 客户端
- Windows 主控 -> macOS 客户端
- macOS 主控 -> Windows 客户端
- 同一局域网下至少覆盖一组有线网络和一组无线网络

MVP 平台前提：

- 用户已登录普通桌面会话
- 已授予所需输入权限
- 不把 UAC、锁屏、登录前会话纳入验收

## 3. MVP 功能测试

### 3.1 设备发现、添加与会话

- 新设备自动发现成功
- 手工添加设备成功
- 首次配对需要确认
- 配对后建立受信关系
- 已配对设备自动重连
- 撤销信任后无法继续连接

### 3.2 屏幕拓扑

- 新配对设备进入待布局区
- 拖拽到上、下、左、右后邻接关系正确
- 多设备布局无重叠
- 非法布局被阻止
- 布局保存后重启仍生效
- 设备离线后占位保留
- 布局变更后运行时立即生效

### 3.3 键盘和鼠标共享

- 主控端输入可以切换到目标设备
- 边界切屏方向与拓扑一致
- 键盘输入无明显丢键
- 鼠标移动无明显跳跃
- 切换回主控端行为正常

### 3.4 文本剪贴板共享

- 文本双向同步
- 关闭剪贴板共享后不再同步
- 循环同步被抑制

### 3.5 常驻与权限引导

- Windows 托盘可正常常驻
- macOS 菜单栏或常驻图标可正常工作
- 关闭窗口后后台持续运行
- 重新打开主窗口正常
- 缺失权限时可以看到明确引导

## 4. Post-MVP 扩展测试

### 4.1 图片剪贴板

- 图片双向同步
- 跨平台图片格式转换可用

### 4.2 文件传输

- 单文件发送成功
- 多文件发送成功
- 接收端确认后开始传输
- 接收端拒绝后任务终止
- 中途取消有效
- 文件名冲突处理符合预期

### 4.3 鼠标灵敏度

- 默认配置下行为接近原始输入
- 灵敏度倍率生效
- 不同目标设备配置独立生效

### 4.4 滚轮平滑

- 默认关闭时行为正常
- 开启后滚轮更平滑
- 调整倍率和参数后行为可见变化
- 无明显失控和拖尾过长问题

## 5. 性能测试

### 5.1 MVP 性能门槛

- 同一局域网条件下，键鼠输入端到端延迟中位数不高于 20 ms
- 同一局域网条件下，键鼠输入端到端延迟 p95 不高于 35 ms
- 60 秒高频鼠标移动测试中，不出现持续超过 100 ms 的明显卡顿
- 拓扑热更新期间现有连接不中断
- 拓扑热更新导致的输入停顿不高于 500 ms
- 空闲状态下，桌面壳与核心进程合计平均 CPU 占用应低于 5%

### 5.2 Post-MVP 性能门槛

- 图片剪贴板同步不应导致输入链 p95 延迟退化超过 10%
- 文件传输进行时，输入链 p95 延迟退化不应超过 20%
- 高速滚动下无明显队列堆积

### 5.3 资源占用记录

- 空闲状态 CPU/内存
- 连接状态 CPU/内存
- Post-MVP 文件传输期间 CPU/内存

## 6. 安全测试

- 未配对设备连接应被拒绝
- 指纹或身份不匹配应被拒绝
- 撤销信任后旧会话失效
- 文本剪贴板不得对未授权设备开放
- 证书变化后旧信任关系不得自动继承

## 7. 打包验证

### Windows

- 安装包可安装
- 安装后应用可运行
- 权限引导正确
- 卸载流程正常

### macOS

- DMG 可挂载和安装
- 首次运行权限引导正常
- 常驻行为正常
- 签名与后续公证流程可验证

## 8. 完成标准

任务只有在满足以下条件时才能标记完成：

- 对应范围内的功能测试通过
- 相关性能测试通过或记录不适用
- 无阻断缺陷
- 变更后的设计文档与 `task.json` 已同步
