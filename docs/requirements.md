# Deskflow-Plus 需求文档

## 1. 产品目标

Deskflow-Plus 是一款面向局域网场景的 Windows 和 macOS 双平台桌面应用。产品目标是在参考 Deskflow 核心使用模型的基础上，提供更易维护、更易扩展、更安全的多机输入共享体验，并在 MVP 稳定后逐步扩展增强能力。

长期目标：

- 多台设备共享一套键盘和鼠标
- 局域网内低延迟控制
- 安全配对后建立长期信任关系
- 配对后通过拖拽配置主副屏幕相对位置
- 支持文本和图片剪贴板共享
- 支持应用内文件发送
- 支持鼠标灵敏度调节
- 支持滚轮平滑调节

## 2. 发布策略

- MVP 先交付“配对 + 会话 + 拓扑 + 键鼠 + 文本剪贴板 + 常驻壳”
- 图片剪贴板、文件传输、鼠标灵敏度、滚轮平滑属于 Post-MVP 增强
- 所有文档、测试和任务计划均以 MVP 口径作为当前里程碑标准

## 3. 平台范围

- 支持平台：Windows、macOS
- 不支持平台：Linux

## 4. 使用模式

- 单主控、多客户端
- 主控端负责屏幕拓扑编排和输入转发
- 客户端接收控制、同步剪贴板、接收后续扩展能力的数据流

## 5. MVP 功能需求

### 5.1 设备发现与添加

- 支持局域网内自动发现设备
- 自动发现可失败，但手工输入 IP 或主机名添加必须始终可用
- 发现结果需要展示设备名、平台、连接状态、受信状态

### 5.2 安全配对与信任管理

- 每台设备首次启动时生成唯一设备身份和本地证书
- 首次连接必须进行配对确认
- 配对确认需要展示可比对的设备指纹
- 配对成功后建立受信关系
- 受信设备信息本地持久化
- 未配对设备不得控制输入或访问剪贴板
- 支持撤销信任设备
- 设备证书变化后必须重新配对

### 5.3 屏幕拓扑配置

- 配对成功后，新设备进入待布局区
- 主控端必须提供拖拽式屏幕拓扑配置界面
- MVP 采用固定网格模型，默认主控设备位于中心区域
- 用户可以通过拖拽指定副屏位于主屏的上、下、左、右
- 支持多个副屏之间的相对布局
- 布局必须可持久化
- 布局变更后必须实时生效
- 不允许重叠布局
- 不允许形成孤立布局
- 不允许不存在边界接壤的无效邻接
- 设备离线后保留布局占位并标记离线

### 5.4 键盘和鼠标共享

- 主控端捕获本地输入事件
- 当光标到达布局边界时切换控制目标设备
- 目标设备需接收并注入键盘和鼠标事件
- 默认行为应保持输入平滑、无明显卡顿
- 输入链优先级高于其他业务流
- 输入链实现不依赖 GUI 渲染线程

### 5.5 文本剪贴板共享

- MVP 仅支持文本剪贴板共享
- 剪贴板变化需在已连接受信设备间同步
- 需要防止循环同步
- 支持用户关闭剪贴板共享

### 5.6 应用运行方式

- 提供 GUI 设置界面
- 提供后台常驻核心进程
- Windows 侧支持托盘
- macOS 侧支持菜单栏或常驻图标
- 首次运行需要提供权限引导与失败提示

## 6. Post-MVP 功能需求

### 6.1 图片剪贴板共享

- 支持图片双向同步
- 支持跨平台图片格式转换
- 不影响文本剪贴板路径的稳定性

### 6.2 文件传输

- 仅支持应用内文件发送，不支持系统级拖拽到远端桌面
- 支持单文件和多文件发送
- 接收端默认需要确认
- 支持显示传输进度
- 支持取消传输
- 支持处理文件名冲突
- 文件传输不得阻塞键鼠控制

### 6.3 鼠标灵敏度调节

- 支持按目标设备配置鼠标灵敏度
- 支持基础倍率调节
- 预留曲线参数扩展能力
- 默认配置下行为需接近原始输入

### 6.4 滚轮平滑调节

- 支持按目标设备配置滚轮平滑
- 支持总倍率调节
- 支持平滑开关
- 预留平滑时长和衰减参数扩展
- 默认配置下不得引入明显失控或拖尾

## 7. 非功能需求

- 仅面向局域网直连场景
- 体验优先
- 输入链低延迟优先
- 安全默认开启
- 自动发现失败不能阻止手工添加
- 核心配置、状态和错误必须可诊断
- 所有核心功能需具备功能测试和性能测试

## 8. 权限与平台约束

### Windows

- 需要满足输入捕获与输入注入所需权限条件
- 需要处理后台常驻、托盘、开机启动等平台行为
- MVP 仅覆盖用户已登录的普通桌面会话
- MVP 不覆盖 UAC 提权窗口、锁屏界面和登录屏幕

### macOS

- 需要处理 Accessibility 权限
- 需要处理 Input Monitoring 权限
- 需要处理后台常驻和菜单栏行为
- 需要检测 Secure Input 等导致输入捕获受限的场景并提示用户
- MVP 不覆盖登录前会话和系统级特权注入

## 9. MVP 不包含范围

- Linux 支持
- 图片剪贴板共享
- 应用内文件发送
- 鼠标灵敏度调节
- 滚轮平滑调节
- 系统级拖拽传文件到远端桌面
- 共享目录同步
- 外网穿透
- 云中继
- 富文本全格式透传
- 任意像素级异形拓扑拼接
